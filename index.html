Privacy Policy

Effective Date: July 15, 2026

This Privacy Policy outlines how our internal integration application ("the App") handles data when connecting to the Pinterest API. The App operates strictly as a personal utility tool to manage internal content queues and does not offer public commercial services.

1. Data We Access and Collect

Through the official Pinterest API, the App requests permissions to access specific data points necessary to execute user-initiated actions. This data includes:

  • Access Tokens: Temporary cryptographic keys used to securely authenticate your account via OAuth.
  • Board Identifiers: Metadata names and ID strings of your boards to allow correct target selection for Pin delivery.
  • Content Assets: Title strings, text descriptions, and destination landing page URLs provided within the code execution scope to create your Pins.

2. How Data is Used

Any data accessed via the Pinterest API is utilized strictly to format, compile, and publish graphical Pin layouts directly onto the user's personal Pinterest boards. The application performs no background operations outside of explicit execution scripts.

3. Data Sharing, Retention, and Third-Party Policies

  • No Data Sharing or Sales: We do not sell, rent, trade, or distribute any user data, tokens, or analytics profiles gathered through the Pinterest API to third-party ad networks or data brokers.
  • Ephemeral Processing: Access tokens and data values are processed purely in ephemeral cache memory during runtime environment cycles (e.g., within private GitHub Actions execution boundaries) and are not permanently saved or aggregated on external servers.
  • Third-Party Libraries: The workflow utilizes image translation tools (ImgBB API) to generate short-lived image URLs required by the Pinterest ingestion engine. These assets are configured to automatically self-delete post-execution.

4. User Control and Revocation

You maintain full authority over your data profile. You can instantly cut the application's line of connection and revoke its active API credentials at any time by navigating to your native account settings inside the Pinterest platform under Settings ➔ Apps and Websites.

5. Contact

For questions regarding this operational compliance policy layout, please open an administrative issue ticket within the project's source code repository portal.
